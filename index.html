Welcome to Simplifie ColdIgniter.

Hello PHP, Git and open-source experts out there using CodeIgniter. We're doing a free PHP framework. Just a hobby. Won't be big and professional --like PHP and Rails at Ruby On Rails. This has been brewing since April and we're hoping for it to get ready this year. We'd like any help, feedback on things people like or missing features on CodeIgniter, as our framework is based on top of it. Any suggestions are welcome --but we can't promise we'll actually implement them. -Excerpt from Linus.

The project includes extra simple features that were missing in CodeIgniter 2.1.0 to make developers more productive by speeding up workflow and meet deadlines.

Designer Templates

ColdIgniter is designed to be integrated to Foundation Framework 5 for its view files because they're cooler and not boring to look at in the long run! But okay, let's integrate Bootstrap soon as well.

Purpose

The goal is to create a framework beyond CodeIgniter, but below CMS, that the original authors won't allow to be implemented for valid reasons. We believe it's still possible to maintain their concepts while adding more tools so that developers don't need to look for libraries elsewhere.

To do this, we need to focus on CLI --but one that doesn't need configurations. Here's a generator example using either MS-DOS or Mac Terminal inside the ColdIgniter folder:

php index.php g crud student id:int full_name:varchar user:ref //This will generate CRUD files.

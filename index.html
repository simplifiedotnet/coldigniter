### Welcome to Simplifie ColdIgniter.

Hello PHP, Git and open-source experts out there using CodeIgniter. We're doing a free PHP framework. Just a hobby. Won't be big and professional --like PHP and Rails at Ruby On Rails. This has been brewing since April and we're hoping for it to get ready this year. We'd like any help, feedback on things people like or missing features on CodeIgniter, as our framework is based on top of it. Any suggestions are welcome --but we can't promise we'll actually implement them. -Excerpt from Linus.

Designer Files

ColdIgniter is currently integrated to Foundation Framework 5 for its view files. Bootstrap will be included soon.

Purpose

Include extra features missing in CodeIgniter 3 to make developers more productive by speeding up workflow and meet project deadlines. The goal is to create another framework on top of CodeIgniter with lots of integrated libraries and helpers using Composer so developers don't need to look elsewhere.

Here's a generator example using either MS-DOS or Mac Terminal inside the ColdIgniter folder:

php index.php gexample crud student id:int full_name:varchar //Generates CRUD files.

Here's a video that shows the early structure of the generator https://www.youtube.com/watch?v=xW_o5flX3WU.

Other great features include:

  • phpDocumentor
  • Advanced auth with privileges and permissions (not trying to become CMS here)
  • Multiple flavors of PayPal
  • Multiple upload support
  • Gallery
  • Adobe Flash support
  • Multi-threading
  • jQuery UI stuffs such as DateTimePicker
  • Analytics
  • Constrain ratio images
  • Font icons
  • Lots of helpers such as:
    • getPostValuePair() - Retrieves all post values and excludes the ones listed in the array parameter
    • showView(), showJsonView() and showRestResponse
    • validateLoginSession() - Validates controllers and redirects the user depending on the session and passed parameters
    • isLoggedIn() and getLoggedUser()
    • downloadAsCsv()
    • and much more...
